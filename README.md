# HookGen

**AI-powered video editor that transforms a voiceover + raw B-roll into a ready-to-publish 9:16 short-form ad.**

Upload your audio script and B-roll footage. HookGen transcribes the speech, maps it to the AIDA advertising framework, semantically matches every phrase to the right clip, and exports a vertical video — automatically.

## How it works

```
Audio (MP3/WAV)  ──►  Whisper transcription  ──►  Word-level timestamps
                                                         │
                                              Speech rhythm analysis
                                              (detects natural pauses)
                                                         │
B-Roll videos    ──►  Gemini 2.5 Flash  ──►  Segments: description,
(MP4/MOV)             frame-by-frame         mood, keywords, timestamps
                       analysis                          │
                                                         ▼
                                              GPT-4o-mini AIDA analysis
                                              (Attention→Interest→Desire→Action)
                                                         │
                                              Semantic matching via
                                              cosine similarity on
                                              text embeddings
                                                         │
                                                         ▼
                                              FFmpeg 9:16 export
                                              (cut · crop · mux · render)
```

Each speech phrase gets a concrete English visual description generated by GPT-4o-mini. That description is embedded alongside the Gemini video segment descriptions, and the best-matching B-roll clip is selected via cosine similarity — ensuring the visuals always match what the narrator is saying.

---

## Features

- **Phrase-accurate clip selection** — cuts happen at natural speech pauses, not arbitrary intervals
- **AIDA-aware matching** — attention phases show pain points, desire phases show the product in use, action phases show urgency cues
- **Cross-lingual support** — scripts in any language work because visual descriptions are translated to English before embedding
- **AUTO and REVIEW modes** — export instantly or review and approve clips before rendering
- **Real-time progress** — WebSocket updates from transcription through export
- **Product image awareness** — upload a reference product image and Gemini will flag clips that contain it
- **Cascading footage logic** — when a clip runs short, the system chains adjacent segments to fill the slot without gaps

## Getting started

### Prerequisites

- [Docker Desktop](https://www.docker.com/products/docker-desktop/)
- OpenAI API key
- Google Gemini API key

### 1. Clone the repo

```bash
git clone git@github.com:jbangue85/hookgen.git
cd hookgen
```

### 2. Configure environment variables

```bash
cp .env.example .env
```

Edit `.env` and fill in:

```env
OPENAI_API_KEY=sk-...
GEMINI_API_KEY=AIza...
```

### 3. Start the stack

```bash
docker-compose up --build
```

| Service | URL |
|---|---|
| Frontend | http://localhost:3000 |
| Backend API | http://localhost:8000 |
| API docs | http://localhost:8000/docs |

That's it. Open http://localhost:3000, upload your audio and B-roll, and hit **Generate**.

## Pipeline details

The processing pipeline runs as a Celery task chain:

```
transcribe_audio_task
  └─► analyze_video_task  (one per video, sequential)
        └─► match_clips_task
              └─► export_video_task  (AUTO mode only)
```

**Project status flow:**
`created → transcribing → analyzing → matching → exporting → completed`

**Clip selection scoring:**

| Signal | Weight |
|---|---|
| Cosine similarity (phrase ↔ segment) | Base score |
| Keyword overlap | +0.05 per match, max +0.15 |
| AIDA mood match | +0.05 |
| Product visible in attention phase | −0.50 |
| Product visible in interest/desire phase | +0.30 |
| Repetition penalty | −0.25 × use count |
| Adjacency penalty (same clip twice in a row) | −0.30 |

## Local development (without Docker)

**Backend**
```bash
cd backend
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload
# In a separate terminal:
celery -A worker.tasks.celery_app worker --loglevel=info
```

**Frontend**
```bash
cd frontend
npm install
npm run dev
```

**Database migrations**
```bash
cd backend
alembic upgrade head
```
